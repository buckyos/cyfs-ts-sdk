# CYFS SDK 文档

## 安装

### 安装SDK
> - nightly通道 `npm i --save cyfs-sdk-nightly`
> - beta通道 敬请期待
> - stable通道 敬请期待

### 安装CYFS TOOL
> - nightly通道 `npm i -g cyfs-tool-nightly`
> - beta通道 敬请期待
> - stable通道 敬请期待

CYFS TOOL的部分命令需要本机安装CYFS浏览器，或本机是OOD才可使用

CYFS TOOL切换通道安装时，需要在安装命令后加`--force`
